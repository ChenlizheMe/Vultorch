Vultorch

Real-time Torch visualization window with Vulkan zero-copy rendering.

基于 Vulkan 零拷贝渲染的实时 Torch 可视化窗口。

GitHub · Install 安装 · Examples 示例

Overview 概述

Vultorch displays CUDA tensors in a native window by sharing GPU memory between PyTorch and Vulkan. Data stays on the GPU — there is no tensor.cpu() readback and no staging buffer copy. It includes a built-in ImGui interface with sliders, plots, dockable windows, and a 3D scene viewer.

Vultorch 通过 PyTorch 与 Vulkan 共享 GPU 显存，在原生窗口中显示 CUDA 张量。 数据全程留在 GPU 上 — 无需 tensor.cpu() 回读，无中转缓冲拷贝。 内置 ImGui 界面，支持滑条、折线图、停靠窗口和 3D 场景查看器。

  • Zero-copy display via Vulkan external memory interop
  • 通过 Vulkan 外部内存互操作实现零拷贝显示
  • Shared GPU memory — create_tensor() returns a torch.Tensor backed by Vulkan memory (DLPack)
  • 共享 GPU 显存 — create_tensor() 返回由 Vulkan 内存支持的 torch.Tensor（DLPack）
  • Built-in ImGui: sliders, buttons, color pickers, plots, docking layout
  • 内置 ImGui：滑条、按钮、颜色选择器、折线图、停靠布局
  • 3D scene view with orbit camera, MSAA, Blinn-Phong shading
  • 3D 场景查看器，支持轨道相机、MSAA、Blinn-Phong 光照
  • DockBuilder API for programmatic window layout
  • DockBuilder API 支持代码化窗口布局

Usage 用法

Basic visualization 基本用法 

import torch, vultorch
from vultorch import ui

texture = torch.rand(512, 512, 4, device="cuda")

win = vultorch.Window("Viewer", 800, 600)
while win.poll():
    if not win.begin_frame(): continue
    ui.begin("Output")
    vultorch.show(texture)  # zero-copy GPU → screen
    ui.end()
    win.end_frame()
win.destroy()

Zero-copy shared tensor 零拷贝共享张量 

tensor = vultorch.create_tensor(512, 512, channels=4)
tensor[:] = model(input)  # writes are immediately visible on screen

Build from Source 从源码构建

Building from source requires the Vulkan SDK and CMake 3.25+.

从源码构建需要 Vulkan SDK 和 CMake 3.25+。 

git clone --recursive https://github.com/vultorch/vultorch.git
cd vultorch

cmake --preset release-windows
cmake --build --preset release-windows

pip install dist/vultorch-*.whl
